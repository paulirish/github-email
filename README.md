# github-email

Retrieve a GitHub user's email even if it's not public.

![image](https://cloud.githubusercontent.com/assets/39191/7485758/6992dc62-f34f-11e4-9af0-3d0f292f6139.png)


### install
```sh
npm install --global github-email
```

### use
```sh
github-email user 
```
